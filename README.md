# Ajira Mia Moja

Tovuti rasmi ya **Ajira Moja na Mia Moja** — React + Vite (Deploy Web App).

## Endesha lokal

```bash
npm install
npm run dev
```

## Build kwa deploy

```bash
npm run build
```

Folda `dist/` ndiyo unayoweza **upload** au kuunganisha na GitHub kwa Deploy Web App.

- Build command: `npm run build`
- Output directory: `dist`
- Start (preview): `npm start`
